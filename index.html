🔒 安全特性

  • 所有数据仅存储在本地设备
  • API Key 加密保存在本地
  • 白盒加密 + ChaCha20 流密码保护
  • 支持 V1+V2+V3 签名验证
  • 应用锁 PIN 码保护
  • 内容过滤敏感词屏蔽

🛠️ 技术栈

  • 原生 HTML5 + CSS3 + JavaScript
  • 液体玻璃 (Liquid Glass) 设计风格
  • iOS 风格设计
  • OpenAI 兼容 API + SSE 流式响应
  • Web Speech API 语音输入/输出
  • localStorage 本地持久化存储
  • Markdown 实时渲染
